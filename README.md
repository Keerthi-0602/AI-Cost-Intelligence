# AI Cost Intelligence System

## Problem
Organizations lose money due to duplicate invoices and unnoticed cost leakages.

## Solution
This project uses an AI-based agent system to detect duplicate invoices, calculate financial loss, and suggest corrective actions.

## Features
- Detect duplicate invoices automatically
- Calculate potential financial loss
- Suggest corrective actions
- Simple audit output

## Workflow
Data → Detection Agent → Decision → Action → Output

## How to Run
1. Install Python
2. Install pandas:
   pip install pandas
3. Run:
   python main.py

## Impact
- Detects duplicate payments
- Saves approx ₹50,000/month
- Reduces manual audit time by 70%
